# switch

## Props

<!-- @vuese:switch:props:start -->
|Name|Description|Type|Required|Default|
|---|---|---|---|---|
|onChange|改变 switch 状态|`Function`|`false`|() => {}|
|disabled|是否禁用|`Boolean`|`false`|false|
|value|绑定值|`Boolean`|`false`|-|
|label|标签文本|`String`|`false`|-|
